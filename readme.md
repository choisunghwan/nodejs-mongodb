--개발하면서 개념 정리--

서버ㅣ 요청을 하면 요청한 내용을 보내주는 프로그램

서버 요청은 4개 방식이 있음
1. 읽기 (GET)
2. 쓰기 (POST)
3. 수정 (PUT)
4. 삭제 (DELETE)

예를 들어 
' 어떤 사람이 /list 라는 페이지를 GET 요청하면..
거기 해당하는 list.html파일을 보내줌 ' 이라고 코드를 작성하면 서버 개발이라고 할 수 있다.
